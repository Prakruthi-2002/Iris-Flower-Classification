# Iris Flower Classification

## Project Overview
This project uses Machine Learning algorithms to classify Iris flower species based on flower measurements.

## Dataset Information
- Dataset: Iris Dataset
- Total Records: 150
- Classes:
  - Iris-setosa
  - Iris-versicolor
  - Iris-virginica

## Project Workflow
1. Data Loading
2. Data Inspection
3. Data Visualization
4. Data Preprocessing
5. Label Encoding
6. Train-Test Split
7. Model Training
8. Model Evaluation

## Models Used
### Logistic Regression
Used as a baseline classification model.

### K-Nearest Neighbors (KNN)
Distance-based classification algorithm.

### Random Forest
Ensemble learning algorithm used for robust classification.

## Evaluation Metrics
- Accuracy Score
- Confusion Matrix

## Key Findings
- The dataset is clean with no missing values.
- Petal Length and Petal Width are highly important features.
- Machine learning models achieve excellent classification performance.
- Random Forest provides strong predictive capability.
